# 树懒书摘

把微信读书里的划线、想法和书评，变成一个真正可复习、可检索、可写作的个人阅读工作台。

如果你也有这种感觉：微信读书里收藏了很多句子，当时很受触动，过几天就沉进列表里，再也没被重新看见。**树懒书摘** 想解决的就是这件事：让你的书摘不只是“存起来”，而是能被重新遇见、重新组织、重新使用。

## 适合谁

- 微信读书里有大量划线、想法和书评，但很少回看的读者
- 想把读书笔记沉淀成文章、选题、复习材料的人
- 希望拥有一个本地优先、结构清晰的个人书摘库的人
- 不想再在微信读书、文档、笔记软件之间反复复制粘贴的人

## 为什么做它

大多数读书笔记工具都停在“收集”。但阅读真正有价值的部分，往往发生在第二次、第三次回看时。

树懒书摘围绕三个动作设计：

1. **同步**：把微信读书里的书摘集中到本地。
2. **整理**：按书籍、标签、收藏、复习状态组织内容。
3. **再利用**：搜索、复习、导出、AI 问答和写作辅助，让书摘继续产生价值。

## 主要功能

### 微信读书同步

通过微信读书 Skill API Key 导入书籍、划线、想法和书评。导入时支持跳过重复笔记，也可以过滤笔记过少的书，避免书架被零散内容淹没。

### 书籍中心

按书组织所有笔记，快速查看一本书里的划线、想法、章节和收藏内容。适合做单本书复盘，也适合在写作前重新翻材料。

### 温故复习

首页会从你的笔记里挑出值得重看的内容。你也可以进入复习队列，专门处理未复习、收藏或随机笔记。

### 搜索与筛选

支持按书名、作者、正文、想法内容搜索，并结合收藏、未复习、类型等条件缩小范围。找一句曾经划过的话，不再靠记忆硬翻。

### AI 问答与写作

支持 OpenAI、DeepSeek、GLM。你可以围绕一条或多条笔记提问、总结、提炼行动建议，也可以把书摘整理成文章素材。

### 阅读报告

按书生成阅读报告，自动整理主题、高密度章节和精选摘录，帮助你快速回看一本书真正留下了什么。

### 导出中心

支持导出 Markdown、Obsidian、PDF、DOCX、Epub 等格式，方便继续进入你已有的知识库或写作流程。

### 本地优先

应用是 macOS 原生应用。API Key 等敏感信息不会打进安装包，运行时配置保存在你的本机环境中。

## 下载安装

在 Release 页面下载对应版本：

- Apple Silicon Mac：`树懒书摘-<版本号>-arm64.dmg`
- Intel Mac：`树懒书摘-<版本号>-x86_64.dmg`

下载地址：[v1.1 Release](https://github.com/zhouxuan3550/WeRead-Notes-Manager/releases/tag/v1.1)

## 本地开发

```bash
swift build
swift run WeReadNotesManager
```

## 打包

项目提供了可复用的 DMG 打包脚本：

```bash
scripts/package_dmg.sh 1.1 arm64
scripts/package_dmg.sh 1.1 x86_64
```

生成的安装包会输出到 `dist/` 目录。
